\section{Problem statement}
Smartphones are a integrated part of our life, we use it for calling, messaging and surfing and we can use it everywhere. It is hard to use a pointing device to create slide shows while being mobile. To this extent, a more mobile slideshow language will be created. This is a great way to do useful things while “on the go” which is why a new type of “on the go” application is initiated, which is creating slideshows on a mobile device.
